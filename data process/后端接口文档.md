# 战雷遥测后端 接口文档

本文档面向前端开发，说明后端服务对外提供的 HTTP 接口、各接口返回的数据结构与字段含义。

> 所有字段均依据当前源码（`wt_server.py` / `wt_telemetry.py` / `wt_processor.py` / `wt_geo.py` / `wt_proximity.py` / `wt_events.py`）核实。

---

## 一、概述

| 项 | 值 |
|---|---|
| 服务地址 | `http://localhost:8112`（端口可用 `--port` 修改，默认 8112） |
| 数据源 | 游戏本地接口 `http://localhost:8111` |
| 响应格式 | JSON（地图图片接口除外，返回二进制图片） |
| 字符编码 | UTF-8 |
| 跨域 | 已开启 CORS：`Access-Control-Allow-Origin: *`，允许 `GET, OPTIONS` |
| 请求方式 | 全部为 `GET`（另支持 `OPTIONS` 预检） |

### 后端轮询频率（分频多线程）

后端按数据特性分四组、各自独立频率轮询游戏接口并缓存，前端按需取用即可（前端轮询频率与后端解耦）：

| 组 | 默认间隔 | 内容 |
|---|---|---|
| `fast` | 0.1s | `/state`（仪表）+ `/indicators`（座舱）+ 战局状态 |
| `map` | 0.5s | `/map_obj`（地图单位）+ 态势分析 + 接近告警 |
| `events` | 1.0s | `/mission` + `/hudmsg`（击杀/事件）+ `/gamechat`（聊天） |
| `mapimg` | 5.0s | `/map_info`（坐标换算参数）+ 小地图图片 |

各间隔可通过启动参数 `--fast-interval` / `--map-interval` / `--event-interval` / `--mapimg-interval` 调整。

## 二、接口一览

| 路径 | 说明 | 返回 |
|---|---|---|
| `/`、`/health`、`/api/health` | 健康检查 | 见 [健康检查](#六健康检查) |
| `/api/telemetry` | **完整快照**（包含以下所有数据） | 见 [完整快照](#三完整快照-apitelemetry) |
| `/api/state` | 载具仪表状态（飞行/航行参数） | `vehicle` 对象 |
| `/api/indicators` | 座舱原始指示数据 | `indicators` 对象 |
| `/api/map_objects` | 地图单位列表 | `MapObject[]` |
| `/api/map_info` | 坐标换算参数 | `map_info` 对象 |
| `/api/processed` | **自定义加工数据 + 告警** | `processed` 对象 |
| `/api/alerts` | 仅告警（精简版） | `{ level, alerts }` |
| `/api/situation` | 态势（最近敌军/距离方位） | `situation` 对象 |
| `/api/proximity` | 敌军接近告警（边沿触发） | `{ thresholds_m, events }` |
| `/api/kills`、`/api/combat` | 战绩（击杀流 / K-D） | `combat` 对象 |
| `/api/hud` | HUD 事件原始流 | `HudMessage[]` |
| `/api/chat` | 聊天消息 | 原始消息数组 |
| `/api/map.jpg`、`/api/map` | 小地图图片 | 二进制图片（无图时 404 JSON） |

未匹配路径返回 `404`：`{ "error": "not found", "path": "..." }`。

## 三、完整快照 `/api/telemetry`

一次返回所有数据。结构如下（顶层字段）：

```jsonc
{
  "state": "in_battle",          // 连接状态，见枚举
  "timestamp": 1718800000.12,    // 最近一次 fast 轮询的时间戳（秒）
  "in_battle": true,             // 是否在战局
  "domain": "ground",            // 当前载具域，见枚举
  "domain_label": "陆军",        // 载具域中文

  "vehicle": { ... },            // 载具仪表，见 4.1
  "indicators": { ... },         // 座舱数据，见 4.2
  "map_objects": [ ... ],        // 地图单位，见 4.3
  "map_info": { ... },           // 坐标参数，见 4.4
  "player": { ... } | null,      // 地图上“自己”那个单位（MapObject），无则 null
  "enemy_count": 7,              // 敌方单位数
  "ally_count": 5,               // 友方单位数

  "mission_status": "running" | null,
  "mission_objectives": ... | null,
  "hud_events": [ ... ],         // HUD 事件，见 4.8
  "chat": [ ... ],               // 聊天，见 4.9

  "processed": { ... } | null,   // 加工数据+告警，见 4.5
  "situation": { ... } | null,   // 态势，见 4.6
  "proximity": { ... },          // 接近告警，见 4.7
  "combat": { ... } | null,      // 战绩，见 4.10
  "meta": { ... }                // 各轮询组运行状态，见 4.11
}
```

## 四、各结构字段详解

### 4.1 `vehicle`（载具仪表，来源 `/state`）

以飞机字段为主，单位见注释。无数据时 `valid=false`、其余为 `null`。

| 字段 | 类型 | 含义 |
|---|---|---|
| `valid` | bool | 数据是否有效 |
| `altitude_m` | float\|null | 海拔高度（米） |
| `tas_kmh` | float\|null | 真空速（km/h） |
| `ias_kmh` | float\|null | 表速（km/h） |
| `mach` | float\|null | 马赫数 |
| `aoa_deg` | float\|null | 迎角（度） |
| `aos_deg` | float\|null | 侧滑角（度） |
| `load_factor` | float\|null | 过载 Ny |
| `climb_ms` | float\|null | 垂直速度（m/s） |
| `roll_rate_dps` | float\|null | 滚转角速度（度/s） |
| `fuel_kg` | float\|null | 当前燃油（kg） |
| `fuel_full_kg` | float\|null | 满油量（kg） |
| `aileron_pct` / `elevator_pct` / `rudder_pct` / `airbrake_pct` / `flaps_pct` / `gear_pct` | float\|null | 各控制面位置（%） |
| `engines` | EngineState[] | 各发动机数据，见下 |
| `raw` | object | 原始 JSON（陆战/其它字段从此取） |

**EngineState**（`engines[]` 元素）：`index`、`throttle_pct`、`power_hp`、`rpm`、`manifold_pressure_atm`、`oil_temp_c`、`thrust_kgs`、`efficiency_pct`。

### 4.2 `indicators`（座舱数据，来源 `/indicators`）

| 字段 | 类型 | 含义 |
|---|---|---|
| `valid` | bool | 是否有效 |
| `army` | string\|null | 兵种：`air` / `tank` 等 |
| `vehicle_type` | string\|null | 载具代号，如 `ki_61_1a_otsu_china` |
| `speed` | float\|null | 速度（m/s） |
| `heading_deg` | float\|null | 罗盘航向 |
| `roll_deg` / `pitch_deg` / `bank_deg` | float\|null | 姿态角 |
| `vario_ms` | float\|null | 升降速率（m/s） |
| `g_meter` / `g_meter_min` / `g_meter_max` | float\|null | 过载表当前/最小/最大 |
| `throttle` | float\|null | 油门（>1 表示 WEP/加力） |
| `flaps` / `gears` / `airbrake` | float\|null | 襟翼/起落架/减速板 |
| **直升机** | | |
| `is_helicopter` | bool | 是否直升机 |
| `prop_rpm` | float\|null | 旋翼/螺旋桨转速 |
| `radio_altitude` | float\|null | 无线电高度（离地） |
| **发动机类型与温度** | | |
| `is_jet` | bool | 是否喷气发动机 |
| `water_temperature` | float\|null | 水温℃（仅液冷活塞机） |
| `head_temperature` | float\|null | 缸头温度℃（仅气冷活塞机） |
| `turbine_temperature` | float\|null | 涡轮/排气温度℃（仅喷气机） |
| `oil_temperature` | float\|null | 油温℃ |
| **地面载具（坦克）** | | |
| `gear` | float\|null | 档位 |
| `gear_neutral` | float\|null | 空挡档位编号 |
| `rpm` | float\|null | 发动机转速 |
| `stabilizer` | float\|null | 火炮稳定器（1=启用） |
| `first_stage_ammo` | float\|null | 一级待发弹数 |
| `crew_total` / `crew_current` | float\|null | 乘员总数/存活数 |
| `cruise_control` | float\|null | 定速巡航档 |
| `lws` | float\|null | 激光告警（-1=无设备/未被照射） |
| `ircm` | float\|null | 红外对抗 |
| `has_speed_warning` | float\|null | 超速/转向告警 |
| `gunner_state` / `driver_state` | float\|null | 炮手/驾驶员状态 |
| `raw` | object | 原始 JSON |

### 4.3 `map_objects`（地图单位，`MapObject[]`）

坐标均为 0~1 归一化值（x 向右=东，y 向下=南）。

| 字段 | 类型 | 含义 |
|---|---|---|
| `type` | string | 大类：`aircraft` / `ground_model`（含坦克与军舰）/ `airfield` 等 |
| `icon` | string | 图标/兵种：`Player` / `Fighter` / `MediumTank` / `SPAA` 等 |
| `faction` | string | 阵营：`self` / `ally` / `enemy` / `other`（按颜色推断） |
| `color_hex` | string | 颜色 `#RRGGBB` |
| `color_rgb` | [int,int,int] | RGB |
| `blink` | int | 闪烁标志 |
| `x` / `y` | float\|null | 点坐标（归一化） |
| `dx` / `dy` | float\|null | 朝向单位向量（飞机才有） |
| `sx` / `sy` / `ex` / `ey` | float\|null | 线段端点（机场跑道等） |

### 4.4 `map_info`（坐标换算参数，来源 `/map_info`）

| 字段 | 类型 | 含义 |
|---|---|---|
| `valid` | bool | 是否有效 |
| `map_generation` | int\|null | 地图版本号（换图递增） |
| `grid_size` | [float,float]\|null | 网格参考区域尺寸（**注意：不是整图尺寸**） |
| `grid_steps` | [float,float]\|null | 网格格距（米） |
| `grid_zero` | [float,float]\|null | 网格原点（米） |
| `map_min` / `map_max` | [float,float]\|null | 地图世界边界（米）。**真实距离换算用 `map_max - map_min`** |
| `raw` | object | 原始 JSON |

### 4.5 `processed`（加工数据 + 告警，核心）

后端基于机型配置（`vehicle_profiles.json`）计算的派生信息与告警。

| 字段 | 类型 | 含义 |
|---|---|---|
| `vehicle_type` | string\|null | 载具代号 |
| `profile_matched` | bool | 是否在配置表中匹配到该机型（false=用默认值） |
| `army` | string\|null | 兵种 |
| `vehicle_class` | string | 归类：`air` / `ground` / `heli` / `unknown` |
| `fuel_fraction` | float\|null | 剩余燃油比例 0~1 |
| `fuel_kg` | float\|null | 当前燃油 |
| `fuel_burn_rate_kgs` | float\|null | 估算油耗（kg/s） |
| `fuel_remaining_sec` | float\|null | 估算剩余可飞时间（秒） |
| `afterburner_active` | bool | 是否开加力 |
| `afterburner_elapsed_sec` | float | 连续加力时长（秒） |
| `afterburner_max_sec` | float\|null | 加力时间上限（0/null=不限） |
| `ias_kmh` / `aoa_deg` / `altitude_m` | float\|null | 表速/迎角/高度 |
| `g_now` / `g_max` / `g_min` | float\|null | 当前/本架次峰值/谷值过载 |
| `crew_current` / `crew_total` | float\|null | 坦克乘员存活/总数 |
| `ammo_first_stage` | float\|null | 一级待发弹 |
| `gun_stabilizer` | bool\|null | 火炮稳定器 |
| `gear_position` | int\|null | 档位：>0 前进/0 空挡/<0 倒车 |
| `rotor_rpm` | float\|null | 旋翼转速（直升机） |
| `radio_altitude_m` | float\|null | 离地高度（直升机） |
| `vario_ms` | float\|null | 升降速率（直升机） |
| `water_temp_c` | float\|null | 水温（液冷活塞） |
| `head_temp_c` | float\|null | 缸头温度（气冷活塞） |
| `turbine_temp_c` | float\|null | 涡轮温度（喷气） |
| `oil_temp_c` | float\|null | 油温 |
| `alerts` | Alert[] | 告警列表，见下 |
| `flags` | object | 各告警代码 -> 是否触发（bool 映射） |
| `level` | string | 所有告警的最高等级：`info` / `warning` / `critical` |

**Alert**（`alerts[]` 元素）：

| 字段 | 类型 | 含义 |
|---|---|---|
| `code` | string | 告警代码，见下表 |
| `level` | string | `info` / `warning` / `critical` |
| `message` | string | 中文提示文本 |
| `value` | float\|null | 相关数值 |

**告警代码（`code`）**：

| code | level | 含义 |
|---|---|---|
| `fuel_low` / `fuel_critical` | warning / critical | 燃油偏低 / 告急 |
| `afterburner_warning` / `afterburner_overrun` | warning / critical | 加力将达上限 / 加力超时 |
| `stall_warning` / `stall_critical` | warning / critical | 失速 |
| `aoa_high` / `aoa_critical` | warning / critical | 迎角过大 |
| `altitude_low` / `altitude_critical` | warning / critical | 高度过低 |
| `engine_overheat` / `engine_overheat_critical` | warning / critical | 发动机过热（水温/缸头/涡轮，消息文本标明） |
| `oil_overheat` / `oil_overheat_critical` | warning / critical | 机油过热 |
| `vrs_warning` / `vrs_critical` | warning / critical | 直升机涡环状态 |
| `crew_loss` / `crew_critical` | warning / critical | 坦克乘员阵亡 / 濒临阵亡 |
| `ammo_low` / `ammo_empty` | info / warning | 一级弹药偏少 / 耗尽 |
| `laser_warning` | critical | 遭激光照射（可能被锁定/测距） |

> 告警为**状态型**：只要条件持续满足，每帧都会出现在 `alerts` 中，前端可自行去重展示。

### 4.6 `situation`（态势，来源态势分析）

以“自己”为中心，输出敌军距离方位。距离已修正为真实米值。

| 字段 | 类型 | 含义 |
|---|---|---|
| `has_player` | bool | 是否找到自己 |
| `enemy_count` / `ally_count` | int | 敌/友数量 |
| `player` | object\|null | `{ x, y, heading_deg, grid }` |
| `nearest_enemy` | object\|null | 最近敌军（结构同 `enemies[]`） |
| `nearest_air_threat` | object\|null | 最近敌机 |
| `air_threat_count` | int | 敌机数量 |
| `enemies` | array | 全部敌军，按距离升序 |

**`enemies[]` 元素**：

| 字段 | 类型 | 含义 |
|---|---|---|
| `icon` | string | 图标 |
| `type` | string | 大类 |
| `category` | string\|null | **中文兵种类别**（见枚举） |
| `x` / `y` | float | 归一化坐标 |
| `distance_m` | int\|null | 距离（米） |
| `bearing_deg` | float | 绝对方位角（正北=0，顺时针） |
| `relative_deg` | float\|null | 相对自身航向（-180~180，负左正右） |
| `heading_deg` | float\|null | 该单位航向（飞机才有） |

### 4.7 `proximity`（敌军接近告警，边沿触发）

```jsonc
{
  "thresholds_m": { "vs_air": 3000, "vs_ground": null },  // 当前生效的告警距离；null=该类不告警
  "events": [ ... ]   // 触发的接近事件流（累积，上限 100 条）
}
```

**接近事件（`events[]` 元素）**：敌军**首次进入**告警距离时触发一次（持续在范围内不重复）。

| 字段 | 类型 | 含义 |
|---|---|---|
| `id` | int | 事件序号（递增） |
| `ts` | float | 时间戳（秒） |
| `kind` | string | `enter`=从范围外进入 / `appear`=范围内新出现 |
| `icon` | string | 图标 |
| `type` | string | 大类 |
| `category` | string\|null | 中文兵种类别 |
| `is_air` | bool | 是否空中目标 |
| `distance_m` | int\|null | 进入时距离（米） |
| `bearing_deg` | float\|null | 绝对方位角 |
| `compass` | string\|null | 八方位中文（北/东北/…） |
| `relative_deg` | float\|null | 相对航向角 |
| `clock` | int\|null | 时钟方位（12=正前，3=右，6=后，9=左） |
| `threshold_m` | int\|null | 触发时所用的距离阈值 |

> 告警距离随**我方兵种 × 敌方类型**变化：空军/直升机仅对空中敌人告警；陆军/海军仅对地面/水面敌人告警。详见 `vehicle_profiles.json` 的 `_proximity`。

### 4.8 `hud_events`（HUD 事件，`HudMessage[]`）

| 字段 | 类型 | 含义 |
|---|---|---|
| `id` | int | 事件 id |
| `kind` | string | `event` / `damage` |
| `msg` | string | 文本 |
| `sender` | string | 发出方 |
| `enemy` | bool | 是否敌方 |
| `mode` | string | 模式 |
| `time` | int\|null | 游戏内时间 |

### 4.9 `chat`（聊天）

游戏 `/gamechat` 原始消息数组，每条含 `id`、`msg`、`sender`、`mode`、`enemy` 等原始字段（增量推送）。

### 4.10 `combat`（战绩，来源击杀解析）

| 字段 | 类型 | 含义 |
|---|---|---|
| `player_name` | string\|null | 玩家名（需启动时配置 `--player-name`） |
| `total_events` | int | 已解析事件总数 |
| `by_action` | object | 各动作计数（动作代码 -> 次数） |
| `leaderboard` | array | 玩家榜 `{ name, kills, deaths }`，前 20 |
| `my` | object\|null | 我的战绩 `{ kills, deaths }`（需配置玩家名） |
| `feed` | KillEvent[] | 击杀流（最新在前） |

**KillEvent**（`feed[]` 元素）：

| 字段 | 类型 | 含义 |
|---|---|---|
| `id` | int | 事件 id |
| `time` | int\|null | 游戏内时间 |
| `action` | string | 动作代码（如 `shot_down` / `destroyed`） |
| `action_label` | string | 中文动作标签 |
| `killer` / `killer_squad` / `killer_vehicle` | string | 施动者 名/战队/载具 |
| `killer_is_ai` | bool | 施动者是否 AI |
| `victim` / `victim_squad` / `victim_vehicle` | string | 受动者 名/战队/载具 |
| `victim_is_ai` | bool | 受动者是否 AI |
| `is_kill` | bool | 是否致命事件 |
| `parsed` | bool | 是否成功解析 |
| `raw` | string | 清洗后的原文 |

### 4.11 `meta`（轮询组运行状态）

每组（`fast` / `map` / `events` / `mapimg`）一条：

| 字段 | 类型 | 含义 |
|---|---|---|
| `interval` | float | 配置的轮询间隔（秒） |
| `count` | int | 已轮询次数 |
| `age_sec` | float\|null | 距上次更新的秒数（null=尚未更新） |

## 五、枚举值参考

### 连接状态 `state`

| 值 | 含义 |
|---|---|
| `offline` | 端口连不上（游戏没开 / 8111 未监听） |
| `not_in_battle` | 能连上但无效（机库/菜单/加载中） |
| `in_battle` | 已进入战局，数据有效 |

### 载具域 `domain` / `domain_label`

| `domain` | `domain_label` |
|---|---|
| `menu` | 主界面/机库 |
| `air` | 空军 |
| `heli` | 直升机 |
| `ground` | 陆军 |
| `naval` | 海军 |
| `unknown` | 未知 |

### 兵种中文类别 `category`

| icon | category |
|---|---|
| `LightTank` / `MediumTank` / `HeavyTank` | 坦克 |
| `TankDestroyer` / `SPG` | 反坦克车 |
| `SPAA` | 防空车 |
| `Wheeled` | 轮式车辆 |
| `Truck` | 运输车 |
| `Structure` | 建筑/工事 |
| 其它地面单位 | 未知地面载具 |
| `Fighter` | 战斗机 |
| `Bomber` | 轰炸机 |
| `Assault` / `Attacker` | 攻击机 |
| 其它飞机 | 未知飞机 |
| 机场/占领区等场景物 | `null` |

### 告警等级 `level`

`info`（无告警）< `warning`（警告）< `critical`（危急）。

## 七、重要说明与已知局限

1. **敌人型号不可知**：游戏只暴露敌人的**兵种类别**（`icon` / `category`），不提供具体型号；具体型号仅自己的载具有（`indicators.vehicle_type`）。
2. **坦克与军舰同为 `ground_model`**：`type` 无法区分二者，需结合 `domain` 判断。
3. **海军无仪表数据**：海军模式下 `/indicators`、`/state` 多为无效，`vehicle`/`indicators` 字段大多为 `null`，但地图/态势/接近告警仍可用。
4. **距离换算**：基于 `map_max - map_min`；若地图无该参数则回退 `grid_size`（可能不准）。
5. **对战类型不可知**：游戏接口不提供街机/真实/全真（AB/RB/SB）信息。
6. **温度告警按发动机类型自动分流**：液冷看水温、气冷看油温+缸头、喷气看涡轮温度；不适用者字段为 `null` 且不告警。
